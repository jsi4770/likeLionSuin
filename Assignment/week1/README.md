### 2. 핵심 내용 요약
- 파이썬의 자료형은 int(정수형), float(실수형), str(문자형)등으로 나뉘며 연산 시 타입 일치가 중요하다.
- 파이썬의 기본 자료 구조
  - 리스트(list)는 순서가 있는 자료형으로 수정이 가능하다.
  - 튜플(tuple)은 순서가 있지만 수정 불가하다.
  - 세트(set)는 순서가 없고 중복을 허용하지 않으며 집합 연산이 가능하다.
  - 딕셔너리(dictionary)는 Key-Value 구조로 되어있다.
- NumPy는 ndarray 기반으로 빠른 연산을 하는 라이브러리이다.
- Pandas는 DataFrame 기반으로 데이터를 불러오고 전처리하는 라이브러리이다.
- 결측치 처리(fillna)는 데이터 분석 시 꼭 해야하는 전처리 과정이다.
- matplotlib과 seaborn은 데이터를 그래프로 표현해 시각화하는 라이브러리이다.
