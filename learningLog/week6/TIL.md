# 📚 6주차 TIL (Today I Learned)

### 1. 오늘 배운 개념 / 주제
- **CNN (Convolutional Neural Network)**: 이미지의 공간 정보를 유지하며 특징을 추출하는 이미지 전문가
- **LLM (Large Language Model)**: 거대 양의 텍스트를 학습해 다음에 올 단어를 예측하는 언어 전문가
- **Transformer**: Attention 메커니즘을 통해 문장에서 어떤 단어가 관련 있는지 파악하는 현대 LLM의 핵심 구조

---

### 2. 핵심 내용 요약

#### **Part 1. CNN (이미지 전문가)**
1.  **필요성**: 일반 신경망은 이미지를 한 줄로 펼치면 위치 정보가 사라지고 파라미터가 폭발하지만, CNN은 이미지를 펼치지 않고 특징을 찾아냄
2.  **핵심 구성 요소**:
    * **합성곱 (Convolution)**: 필터(커널)가 이미지 위를 슬라이딩하며 영역의 특징을 숫자로 요약함
    * **활성화 함수 (ReLU)**: 음수는 0으로, 양수는 그대로 통과시켜 비선형성을 부여하고 복잡한 패턴을 학습함
    * **풀링 (Pooling)**: 가장 큰 값만 뽑는 등의 방식으로 핵심 정보만 남기고 크기를 줄여 연산량을 절감하고 과적합을 방지함
3.  **학습의 깊이**: 얕은 층은 선과 색상을, 깊은 층으로 갈수록 얼굴이나 자동차 같은 전체 형태를 인식함

#### **Part 2. LLM (언어 전문가)**
1.  **핵심 원리**: 입력된 텍스트 뒤에 올 가장 그럴듯한 단어를 예측하는 과정을 반복하여 문장을 완성함
2.  **Transformer (Self-Attention)**: 모든 단어 쌍의 관련도를 계산해 중요한 정보에 더 집중하며 문맥을 파악함
3.  **학습 3단계**:
    * **사전학습 (Pre-training)**: 방대한 데이터를 통해 언어를 이해하는 기본 능력을 갖춤
    * **미세 조정 (Fine-tuning)**: 명령을 따르고 도움이 되는 답변을 하도록 교육함
    * **강화학습 (RLHF)**: 인간의 평가를 반영해 더 안전하고 사람이 원하는 답을 하도록 개선함
4.  **주요 지표**: 단어 처리 단위인 **토큰**, 가중치 개수인 **파라미터**, 한 번에 처리하는 **컨텍스트 윈도우** 등이 있음

---

### 3. 추가 개념 정리

1.  **CNN vs LLM 비교**
    * **CNN**: 이미지(2D)를 다루며 주로 분류, 탐지 작업을 수행함
    * **LLM**: 텍스트(1D 시퀀스)를 다루며 생성, 번역, 질의응답을 수행함
2.  **LLM 사용 시 주의점**
    * **환각 (Hallucination)**: 모르는 것을 그럴듯하게 지어낼 수 있음
    * **최신 정보 부족**: 학습 데이터 마감일(컷오프) 이후의 정보는 알지 못함
    * **해결 방안**: 웹 검색이나 RAG(검색 증강 생성)를 통해 보완 가능함

### 4. 느낀 점 / 다음 목표
- 이미지 처리에 특화된 CNN과 자연어 처리에 혁명을 일으킨 LLM의 구조적 차이를 명확히 이해할 수 있었다.
- 특히 LLM이 단순한 단어 예측을 넘어 문맥을 파악하는 Attention 원리가 인상적이었다.
- 다음에는 LLM의 한계를 극복하는 기술인 **RAG(검색 증강 생성)**나 **Chain of Thought** 기법에 대해 더 자세히 공부해보고 싶다.
