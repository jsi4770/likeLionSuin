### 2. 핵심 내용 요약

1. **머신러닝 vs 딥러닝**
   - 머신러닝: 사람이 특징을 정의
   - 딥러닝: 모델이 특징을 자동 학습

2. **인공신경망**
   - 입력 → 은닉 → 출력 구조
   - 가중치와 활성화 함수로 데이터 처리

3. **텐서**
   - 딥러닝에서 사용하는 데이터 구조 (다차원 배열)
   - PyTorch는 Tensor 기반 연산 수행

4. **배치 사이즈 & 에포크**
   - Batch: 한 번에 학습하는 데이터 수
   - Epoch: 전체 데이터 반복 횟수
   - 과적합 방지를 위해 적절한 설정 필요

5. **학습 과정**
   - 순전파 → 손실 계산 → 역전파 → 가중치 업데이트
   - Loss로 오차 측정, Optimizer로 성능 개선

6. **MNIST 실습**
   - 28×28 이미지 → 숫자 분류
   - 기본 MLP 모델로 약 97% 정확도 달성 가능

---

### 3. 추가 실험 (이번 주 확장 학습)

1. **CNN 모델 적용**
   - `nn.Conv2d`를 활용하여 이미지 특징(선, 곡선 등)을 자동 추출
   - MLP보다 이미지 처리에 더 적합한 구조

2. **학습률 실험**
   - `lr=0.01`, `lr=0.001` 비교
   - 학습률이 크면 빠르지만 불안정, 작으면 안정적이지만 느림

3. **배치 사이즈 실험**
   - `batch_size=16`, `256` 비교
   - 작은 배치: 빠른 업데이트, 불안정
   - 큰 배치: 안정적 학습, 메모리 사용 증가

4. **Fashion-MNIST 적용**
   - `datasets.MNIST → datasets.FashionMNIST` 변경
   - 숫자가 아닌 의류 이미지 분류
   - MNIST보다 난이도가 높아 정확도가 상대적으로 낮음
