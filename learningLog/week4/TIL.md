### 2. 핵심 내용 요약

1. **머신러닝 vs 딥러닝**
   - 머신러닝: 사람이 특징을 정의
   - 딥러닝: 모델이 특징을 자동 학습

2. **인공신경망**
   - 입력 → 은닉 → 출력 구조
   - 가중치와 활성화 함수로 데이터 처리

3. **텐서**
   - 딥러닝에서 사용하는 데이터 구조 (다차원 배열)
   - PyTorch는 모든 데이터를 Tensor로 처리

4. **배치 사이즈 & 에포크**
   - Batch: 한 번에 학습하는 데이터 수
   - Epoch: 전체 데이터 반복 횟수
   - 과도한 epoch → 과적합 위험

5. **학습 과정**
   - 순전파 → 손실 계산 → 역전파 → 가중치 업데이트
   - Loss: 오차 측정
   - Optimizer: 성능 개선 (Adam 많이 사용)

6. **MNIST 실습**
   - 28×28 이미지 (입력 784)
   - 간단한 신경망으로 숫자 분류
   - 약 97% 정확도 달성 가능
